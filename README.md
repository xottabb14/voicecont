# voicecont

"Маленький" пример того, как можно реализовать голосовое управление на PC под управлением ОС Windows.

##### Возможности скрипта:
* Запуск/закрытие программ
* Озвучивание событий
* Управление окнами посредством автоматического нажатия горячих клавиш
* Набор текста голосом
* Управление медиаплеером

##### Требования:
* Наличие Python3 на PC
* Наличие микрофона
Библиотеки: 
 * **pip install SpeechRecognition**
 * **pip install pywin32**
 * **pip install PyAutoGUI**
 * **pip install pywinauto**

Запуск производится с помощью bat-файла: **voice.bat**

###### Голосовые команды:
* Запусти голосовое управление
* Отключи голосовое управление
* Окно влево
* Окно вправо и т.д.
* Запусти блокнот, закрой блокнот
* Включи/выключи режим печати

###### Полный список команд можно найти в комментариях кода.

##### P.S. В файле **sinonims.py** можно задать синонимы к открываемым программам. В файле **pathscmd.py** пути к ним. Файл **config.py** содержит синонимы к командам.
